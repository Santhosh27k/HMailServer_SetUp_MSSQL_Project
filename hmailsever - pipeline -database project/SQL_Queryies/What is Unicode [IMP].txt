In SQL Server, Unicode strings are character strings that can store characters from multiple languages ‚Äî including symbols, emojis, and non-Latin scripts (e.g., Hindi, Chinese, Arabic).

They are stored using 2 bytes per character, unlike non-Unicode strings (which usually use 1 byte per character and are limited to one code page / language).
